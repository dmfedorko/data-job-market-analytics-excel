# Data Job Market Analytics via Excel

This repository contains two analytical files based on a dataset of tech job postings sourced from Luke Barousse's *Excel for Data Analytics* course.

### 1. **Dashboard**
An interactive visualization of key insights from the job postings dataset, including salary distributions, role comparisons, and schedule type frequency patterns across different countries.
